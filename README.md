# Is it Sunny in Ireland? ☀️

Find sunny weather anywhere on the island of Ireland for the next 7 days.

## Features
- Check weather for 400+ (currently 50) Irish locations
- Sort results by distance from your location
- 3-hour caching to respect API limits
- Powered by Met Éireann data
- Includes weather warnings required by Met Éireann to use API
